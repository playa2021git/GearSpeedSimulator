# GearSpeedSimulator
中学校技術科向けの歯車回転速度シミュレータです。
歯車の歯数や回転数を変更しながら、駆動輪と被動輪の回転速度の関係を視覚的に学習できます。

## 概要

この教材では、Three.jsを用いて歯車の回転を3D表示します。
歯数の違いによって回転速度がどのように変化するか、また、平歯車で噛み合う場合とチェーンで伝達する場合で回転方向がどう変わるかを確認できます。

## 主な機能

* 駆動輪の回転数を変更
* 歯車の歯数を変更
* 平歯車による回転伝達の表示
* チェーン駆動による回転伝達の表示
* 駆動輪に対して被動輪が何倍速、または何分の一倍速になるかを表示
* 回転方向の違いを視覚的に確認
* Three.jsによる3D表示

## 想定する授業での使い方

中学校技術・家庭科（技術分野）の「エネルギー変換の技術」や、機械の仕組みを扱う授業での利用を想定しています。

特に、以下の内容を確認する教材として使えます。

* 歯数が多い歯車は回転が遅くなること
* 歯数が少ない歯車は回転が速くなること
* 平歯車同士が噛み合うと回転方向が反対になること
* チェーンでつなぐと回転方向が同じになること
* 複数の歯車を組み合わせると速度比が変化すること

## 注意

このシミュレータは教育用教材です。
歯形やチェーンの動きは、学習内容を理解しやすくするために一部簡略化しています。
工業設計や機械設計の厳密な計算に使用することは想定していません。
